# updates

هذا المجلد مخصص لأرشيفات التحديث المضغوطة.

## الرفع من GitHub

- ادخل إلى مجلد `updates` في GitHub.
- اختر `Add file` ثم `Upload files`.
- ارفع ملف ZIP الخاص بالتحديث.
- بعد الرفع على فرع `main`، سيقوم GitHub Actions بفك الضغط تلقائياً وتطبيق الملفات على جذر الموقع.
- يتم نقل ملف ZIP بعد تطبيقه إلى `updates/applied` حتى لا يتكرر تطبيقه.

## الرفع من لوحة الإدارة

- ارفع ملف ZIP من تبويب `التحديثات` داخل لوحة الإدارة.
- يتم تطبيق الملفات تلقائياً على جذر الموقع.

## ملاحظات

- إذا كان ملف ZIP يحتوي على مجلد واحد داخله، سيتم تطبيق محتويات هذا المجلد مباشرة على جذر الموقع.
- يتم تجاهل `.git` و`.github` و`node_modules` و`updates` أثناء التطبيق.
- إذا كان التحديث يغير ملفات السيرفر مثل `server.js` أو `package.json` فقد تحتاج إعادة تشغيل السيرفر.
